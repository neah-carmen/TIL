## testing

you will use the `rspec` gem to test your Ruby code

```ruby
RSpec.describe Class do
  describe "#method" do
    it "the code that the method should execute" do
      code = Class.new
      result = code
      expect(result).to eq(value)
    end
  end
end

```

and then in your terminal, to run rspec

```bash
rspec code.rb
```
